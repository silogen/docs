# Document Collection Service

The document collection service allows you to maintain your vector indices which you can use as part of Retrieval Augmented Generation.
The API offers functionality to manage the documents in your collection, and to query the collection for documents matching a given query.

You can find documentation about the accessing the corresponding functionality via our Developer Console [here](../developer-console/collections.mdx).

The service base url is: https://document-collection.services.silogen.ai/

The Open API documentation is here:

https://document-collection.services.silogen.ai/docs

https://document-collection.services.silogen.ai/redoc
