Datasets

Silogen allows your to create and manage datasets in the platform for Fine-tuning and Evaluation purposes. Datasets are a collection of data points, where each datapoint represents a conversation, with or without the models' expected response, depending on the use-case.

Uploading the dataset​

Navigate to the Datasets page of the developer console to upload the dataset. You can provide a name and description for the dataset, and upload a file representing the dataset in JSONL format. The type of dataset should be selected when uploading the file, and the format of each row should correspond to the type of dataset selected.

Data formats​

Depending on the selected dataset type, each row in the JSONL file should be formatted as below

Fine-tuning​

{"messages":[{"role":"system","content":"You are a helpful Geography bot. Only answer questions about Geography."},{"role":"user","content":"What is the capital of France?"},{"role":"assistant","content":"Paris"}]}

Evaluation​

{"messages":[{"role":"system","content":"You are a helpful Geography bot. Only answer questions about Geography."},{"role":"user","content":"What is the capital of France?"}],"response":"Paris"}

Direct Preference​

{"prompt_messages":[{"role":"system","content":"You are a helpful Geography bot. Only answer questions about Geography."},{"role":"user","content":"What is the capital of France?"}],"chosen_messages":[{"role":"assistant","content":"Paris"}],"rejected_messages":[{"role":"assistant","content":"I do not know"}]}

Archiving datasets​

If you have identified a dataset that is no longer relevant to your use cases, you can choose to archive that dataset, by selecting the dataset(s) and clicking the "Archive" button at the top of the screen.

Un-archiving datasets​

To subsequently un-archive a dataset, you can select the "Archived" filter option, select the dataset(s) you would like to un-archive and click the "Un-archive" button at the top of the screen.
